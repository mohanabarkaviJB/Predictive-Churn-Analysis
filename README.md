# Predictive Churn Analysis

## Objective
To predict customer churn using Logistic Regression and identify customers who are likely to discontinue the service.

## Dataset
IBM Telco Customer Churn Dataset

## Features Used
- Tenure
- MonthlyCharges
- TotalCharges
- SeniorCitizen

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Methodology
1. Data Collection
2. Data Cleaning
3. Feature Selection
4. Train-Test Split
5. Logistic Regression Model Training
6. Prediction and Evaluation

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## Results
The Logistic Regression model successfully predicted customer churn and achieved good accuracy.

## Conclusion
This project helps businesses identify customers at risk of churn and take proactive retention measures.
